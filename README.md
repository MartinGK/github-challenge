## How to run it

### Install dependencies

I personally use `yarn` but you can use one of them `npm`, `yarn`, `pnpm`, `bun`, Example using `yarn`:

```bash
yarn install
```

### Run the development server

```bash
yarn dev
```

### If you want to run the production server

```bash
yarn build
yarn start
```
